# Posso usar os clubes?

Nosso clube é o próprio Via Horizonte, o qual é excelente, temos piscinas, academias, diversos espaços de lazer, lounge, sala de cinema e mais :)

Temos uma área de lazer completa, vamos usar!

## E o Clube do R1 e R2?

Esses clubes são de uso exclusivo dos proprietários do R1 e R2, não temos acesso.

Contudo, caso conheça algum morador, você pode entrar como convidado – acompanhado pelo proprietário.

Não existe lista de visitantes ou qualquer cadastro do tipo, só entra com o morador, por favor não insista.

## Posso almoçar no restaurante do clube R1?

Sim, pode almoçar lá, funciona a partir das 11 horas, mas não pode usar o clube de forma alguma, por favor respeite.
