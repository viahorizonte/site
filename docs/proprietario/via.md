# VIA

A VIA foi a construtora responsável pelo empreendimento.

O empreendimento foi entregue em 2016.

A VIA já teve até escritório dentro do Via Horizonte, dando um apoio excelente e cuidando da garantia dos apartamentos, fazendo também diversas melhorias no complexo após a entrega.

No momento a VIA está em recuperação judicial e não tem mais escritório no Via Horizonte.

- http://www.grupovia.com.br/recuperacao-judicial

Devido a uma dívida com o Bradesco, em 2019 a VIA repassou cerca de 70 unidades para quitar dívidas com o banco. O banco através de um de seus Fundos de Investimentos revendeu estas unidades, todas já foram vendidas.

### Consigo acionar a via para reparos ou garantia?

No caso da VIA, só é possível acionar via justiça, mas é um processo lento e sem garantias.

A maioria dos novos e antigos moradores arca com os reparos diretamente.
