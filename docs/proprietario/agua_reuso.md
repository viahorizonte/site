# Água de reuso

A área de lazer do Via Horizonte foi projetada para coleta água da chuva.

O piso do parque das crianças e da área externa próxima as churrasqueiras permite a coleta água que é armazenada em uma caixa de 5 mil litros localizada no segundo subsolo (-2).

Essa água é normalmente utilizada para lavagem das garagens.

Estamos tentando ampliar o uso da água de reuso, existem projetos em andamento.
